Terminal Quick Start
# 1. Clona el repositorio e instala Pandoc en tu sistema
git clone https://github.com/MarcoBrenis/Latex-to-EPUB-proyect.git
brew install pandoc # macOS (o tu gestor de paquetes de preferencia)

# 2. Inicia el servidor LaTeX a HTML (Puerto 8084)
cd Latex-to-EPUB-proyect/latex_to_html_converter
python3 latex_to_html.py

# 3. Inicia el servidor HTML a EPUB (Puerto 8081)
cd ../html_to_epub_converter
python3 html_to_epub.py
Desarrollado de forma modular utilizando únicamente la biblioteca estándar de Python 3.
